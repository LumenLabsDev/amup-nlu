# @lumen-labs-dev/node-nlp

> **Deprecated:** Use [`@lumen-labs-dev/basic`](https://www.npmjs.com/package/@lumen-labs-dev/basic) for new projects. See the [v4 Quick Start](../../docs/v4/quickstart.md).

**npm:** [`@lumen-labs-dev/node-nlp`](https://www.npmjs.com/package/@lumen-labs-dev/node-nlp)

Legacy high-level wrapper around v4 packages. It exposes a convenience `NlpManager` API and loads other `@lumen-labs-dev/*` packages on demand. This package remains published for backward compatibility but is no longer the recommended entry point.

## Documentation

- [Quick Start](../../docs/v4/quickstart.md) — recommended starting point
- [NLP plugin](../../docs/v4/nlp-manager.md)
- [NER Manager](../../docs/v4/ner-manager.md)
- [NLU](../../docs/v4/nlu.md)
- [Language Support](../../docs/v4/language-support.md)

## Installation

```bash
npm install @lumen-labs-dev/node-nlp
```

For new work, prefer:

```bash
npm install @lumen-labs-dev/basic
```
